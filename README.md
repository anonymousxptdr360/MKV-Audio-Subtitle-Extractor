# 🇫🇷 Français - 🎬 MKV Audio & Subtitle Extractor

Un outil simple, rapide et intuitif pour extraire facilement les pistes audio et les sous-titres de vos fichiers MKV.

---

# 🛠️ Installation et configuration

Suivez les étapes ci-dessous pour installer et configurer le logiciel.

## 1. Installer Python

Téléchargez puis installez la dernière version de Python depuis le site officiel :

👉 https://www.python.org/downloads/

> **Conseil :** lors de l'installation, pensez à cocher l'option **"Add Python to PATH"**.

---

## 2. Installer CustomTkinter

Ouvrez l'Invite de commandes (**cmd**) puis exécutez l'une des commandes suivantes :

```bash
pip install customtkinter
```

Si cette commande ne fonctionne pas, essayez l'une des suivantes :

```bash
python -m pip install customtkinter
```

ou

```bash
py -m pip install customtkinter
```

---

## 3. Installer MKVToolNix

Téléchargez la **version portable** de **MKVToolNix**, indispensable au fonctionnement du logiciel.

👉 https://mkvtoolnix.download/downloads.html

> **Important :** téléchargez la **version portable**, et non la version avec installateur.

---

## 4. Télécharger le logiciel

Téléchargez la dernière version depuis la page **Releases** du projet :

👉 https://github.com/anonymousxptdr360/MKV-Audio-Subtitle-Extractor/releases

---

# 🚀 Utilisation

1. Lancez le script que vous venez de télécharger.
2. Cliquez sur l'icône **⚙️** située en haut à droite de la fenêtre.
3. Sélectionnez le fichier **mkvmerge.exe**, présent dans le dossier de la version portable de **MKVToolNix**.
4. Une fois cette configuration effectuée, laissez-vous simplement guider par l'interface du logiciel.

C'est tout ! 🎉

# 🇬🇧 English - 🎬 MKV Audio & Subtitle Extractor

A simple, fast, and intuitive tool to easily extract audio tracks and subtitles from your MKV files.

---

# 🛠️ Installation & Setup

Follow the steps below to install and configure the software.

## 1. Install Python

Download and install the latest version of Python from the official website:

👉 https://www.python.org/downloads/

> **Tip:** During installation, make sure to check the **"Add Python to PATH"** option.

---

## 2. Install CustomTkinter

Open the Command Prompt (**cmd**) and run one of the following commands:

```bash
pip install customtkinter
```

If this command doesn't work, try one of these instead:

```bash
python -m pip install customtkinter
```

or

```bash
py -m pip install customtkinter
```

---

## 3. Install MKVToolNix

Download the **portable version** of **MKVToolNix**, which is required for the software to work.

👉 https://mkvtoolnix.download/downloads.html

> **Important:** Download the **portable version**, not the installer.

---

## 4. Download the software

Download the latest version from the project's **Releases** page:

👉 https://github.com/anonymousxptdr360/MKV-Audio-Subtitle-Extractor/releases

---

# 🚀 Usage

1. Launch the script you just downloaded.
2. Click the **⚙️** icon in the top-right corner of the window.
3. Select the **mkvmerge.exe** file located inside the portable **MKVToolNix** folder.
4. Once configured, simply follow the application's interface.

That's it! 🎉
